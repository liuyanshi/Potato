
##执行操作

1. git clone https://github.com/liuyanshi/Potato
2. cd Potato
3. npm install `安装依赖`
4. sudo npm install supervisor -g `它会监视你对代码的改动, 并自动重启应用`
5. 启动方式: 
 1. node app.js 或 supervisor app.js
 2. 重新开一个terminal: 
    1. cd Potato/public/
    2. python -m SimpleHTTPServer 
6. 数据库信息
>MongoDB 存储文章模块,Mysql存储图书模块
